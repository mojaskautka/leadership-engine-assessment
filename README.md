# Leadership Tradeoffs Mini-Assessment

A 15-question leadership assessment that reveals your leadership engine and persona.

## Live Demo

Visit: `https://YOUR_USERNAME.github.io/leadership-assessment/`

## About

This assessment uses a tradeoff-based approach to identify:
- Your primary leadership engine (Direction, Trust, Execution, Development, or Adaptation)
- Your leadership persona within that engine
- Your pressure patterns under stress
- Balance moves to stay centered

## Deployment

This is a single-file React application. Simply enable GitHub Pages on this repository to deploy.
